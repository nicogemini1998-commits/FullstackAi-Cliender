## Para usar la terminal con Claude (cada usuario en su Mac)

1. Clona o descarga el proyecto en tu Mac
2. Ejecuta en la carpeta del proyecto:
   ```
   npm install
   npm run server
   ```
3. Entra a https://fullstackai.cliender.com desde tu navegador
4. Inicia sesión con tus credenciales
5. Añade un nodo Terminal — se conectará automáticamente a tu Claude local

## Notas
- Sesión dura 30 días
- El usuario `nicolas` tiene rol admin
- La terminal conecta a `localhost:3001` (tu Mac) — necesita `npm run server` corriendo
- Si no hay servidor local, la terminal muestra instrucciones de instalación
